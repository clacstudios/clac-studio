## 📥 Installation

### Prerequisites

- **Python 3.8+**
- **PyAudio** (for playback)

### Quick Install

```bash
# Clone or download the repository
git clone https://github.com/clacstudios/clac-studio.git
cd clac-studio

# Install dependencies
pip install pyaudio

# Run the application
python clac-studio.py
